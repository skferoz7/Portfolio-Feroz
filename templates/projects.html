Automated Public Lighting System (IoT)

The Automated Public Lighting System is an IoT-based smart street-lighting solution designed to minimize energy consumption and reduce manual intervention. The system uses LDR (Light Dependent Resistor) sensors to detect ambient light levels and automatically turn street lights ON during low-light conditions (night/evening) and OFF during daylight.

Key Features:
  • Automatic Light Control: Lights turn ON/OFF based on real-time light intensity.
  • Energy Efficiency: Reduces power wastage by eliminating manual operation.
  • Low Maintenance: Works autonomously without human supervision.
  • Scalable Design: Suitable for highways, campuses, communities, and smart cities.
Student Task Manager

The Student Task Manager is a web-based application designed to help students organize and manage their daily academic tasks efficiently. It allows users to add, track, update, and complete tasks with a simple and clean user interface. The system helps students improve productivity by keeping all tasks in one place with real-time status updates.

Key Features:
  • Task Creation: Add new tasks with title, description, and due date.
  • Task Status Tracking: Mark tasks as Pending, In Progress, or Completed.
  • Edit / Delete: Update task details or remove them anytime.
  • Responsive UI: Clean, simple design optimized for mobile and desktop.
  • User Authentication: Login/Signup system to secure user tasks.
  • Dark/Light Mode: Better reading and smooth user experience.
